# MyBlog 🚀

Welcome to **MyBlog**, a modern, premium blogging platform built with Django.

## ✨ Features
- 📝 **Create & Manage Posts**: Full CRUD functionality for blog posts.
- ❤️ **Like System**: Real-time post liking similar to Instagram/Twitter.
- 💬 **Comments**: Interactive comment sections for every post.
- 👤 **Custom Profiles**: Personalized user bios and profile pictures.
- 🎨 **Premium UI**: Modern design with glassmorphism, gradients, and smooth animations.
- 🔍 **Search**: Powerful search bar to find stories by title, author, or tags.

## 🛠️ Tech Stack
- **Backend**: Django (Python)
- **Frontend**: Bootstrap 5 + Custom CSS
- **Database**: SQLite (Development) / PostgreSQL (Production ready)
- **Styling**: Modern CSS variables, Google Fonts (Inter & Outfit)

## 🚀 Getting Started
1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run migrations: `python manage.py migrate`
4. Start the server: `python manage.py runserver`
